# HsrTicketSnapUpNotice
A program that sends notifications via LINE about Taiwan HSR and Taiwan Railway ticket pre-sale start dates during consecutive holidays.

經由LINE傳送台灣高鐵、台灣鐵路的連續假期疏運車票開賣資訊。
